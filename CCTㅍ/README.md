# CCTV 열람기록 추적 웹앱

일반 시민이 지도에서 주변 CCTV를 확인하고, CCTV별 열람기록과 이상 열람 경고를 볼 수 있는 시연용 웹앱입니다. 발표나 과제 데모에서 바로 실행할 수 있도록 React 화면, 무료 지도, 간단한 백엔드, 파일 기반 샘플 데이터를 한 프로젝트에 넣었습니다.

## 주요 기능

- 로그인 없이 시민용 지도 화면 접속
- OpenStreetMap 기반 무료 지도와 CCTV 마커 표시
- CCTV 상세 정보, 관리 기관, 운영 상태, 최근 열람 요약 표시
- 개인정보를 제외한 공개 열람기록 표시
- 짧은 시간 과다 열람, 야간 열람 증가 경고 표시
- 시민 제보 제출 및 접수 메시지 표시
- 지역명, CCTV 이름, 기관명 검색
- 방범, 교통, 시설관리, 재난안전 목적별 필터
- 관리자 샘플 화면에서 CCTV와 열람기록 등록
- 샘플 CCTV 12개, 열람기록 17개, 제보 2개 포함

## 폴더 구조

```text
cctv-access-watch/
├─ data/
│  └─ db.json          # CCTV, 열람기록, 제보 샘플 데이터
├─ public/
│  ├─ index.html       # React와 지도 라이브러리를 불러오는 화면
│  ├─ app.js           # 실제 화면 기능
│  └─ styles.css       # 디자인 스타일
├─ package.json        # 실행 명령어
├─ server.js           # 백엔드 서버와 API
└─ README.md           # 실행 안내
```

## 실행 방법

1. Node.js를 설치합니다.
   - 이미 설치되어 있으면 이 단계는 건너뛰면 됩니다.
   - 확인 명령어: `node -v`

2. 이 프로젝트 폴더에서 아래 명령어를 실행합니다.

```bash
npm start
```

3. 브라우저에서 아래 주소를 엽니다.

```text
http://localhost:3000
```

## 화면 사용 방법

1. 지도 화면에서 CCTV 마커를 누르면 오른쪽에 상세 정보가 표시됩니다.
2. 검색창에 `명동`, `서울경찰청`, `시청`처럼 입력하면 CCTV가 필터링됩니다.
3. 목적 필터에서 `방범`, `교통`, `시설관리`, `재난안전`을 선택할 수 있습니다.
4. `열람기록 보기`를 누르면 개인 이름 없이 날짜, 목적, 기관명만 표시됩니다.
5. `제보하기`에서 의심스러운 열람이나 사생활 침해 우려를 제출할 수 있습니다.
6. `관리자 샘플`에서 CCTV와 열람기록을 추가하면 데이터 파일에 저장됩니다.

## 초보자용 코드 설명

- `server.js`는 웹페이지를 보여주고 데이터를 주고받는 작은 서버입니다.
- `data/db.json`은 실제 데이터베이스 대신 쓰는 파일입니다. 그래서 별도 DB 설치가 필요 없습니다.
- `public/app.js`는 React로 만든 화면입니다. 검색, 필터, 지도 선택, 제보 제출이 여기에서 동작합니다.
- `public/styles.css`는 화면 색상, 배치, 버튼 모양을 정합니다.
- 지도는 무료 테스트가 가능한 OpenStreetMap과 Leaflet을 사용합니다.

## 참고

이 프로젝트는 시연용 샘플입니다. 실제 서비스로 운영하려면 기관 인증, 열람자 권한 관리, 변경 불가능한 감사 로그, 신고 처리 절차, 개인정보 영향평가 같은 보안 설계가 추가로 필요합니다.
